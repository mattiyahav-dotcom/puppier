# Design Rules

## DO
- Light, minimal, clean, modern design
- Shadcn aesthetic
- Neutral palette using whites, grays, and one subtle warm stone/amber accent color
- Good typography hierarchy and plenty of whitespace
- Font: use Inter from Google Fonts
- Mobile responsive
- Light motion only: scroll animations via IntersectionObserver, smooth transitions, subtle hover states

## DON'T
- No bright saturated gradients
- No gradient text
- No emoji icons — use Lucide icons instead
- No "revolutionary" or "game-changing" marketing copy
- No heavy shadows
- No busy patterns
- No dark mode
- No border radius larger than rounded-2xl
- No cramped spacing
- No tiny fonts smaller than 14px
